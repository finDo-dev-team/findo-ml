# findo-ml

## Installation
1. Cloner le dépôt
```bash
git clone git@github.com:finDo-dev-team/findo-ml.git
```
2. Se placer dans le répertoire du dépôt
```bash
cd findo-ml
```
Créer env python
```bash
python3 -m venv .env
```
Entrer dans cet environement
```bash 
source .env/bin/activate
```
Lancer VSCode (optionnel)
```
code .
```
Installer Flask
```bash
pip install flask
```
